# TODO APP

A simple todo app built using react and typescript.

## Technologies used

Built with:

- HTML
- JS
- CSS
- Reactjs
- TypeScript

## Setup and usage

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
